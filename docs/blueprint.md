# **App Name**: AI Resume Smith

## Core Features:

- Data Input Forms: Allow users to input personal information, work experience, education, skills, and a professional summary.
- AI Content Enhancement: Provide AI-powered suggestions for improving resume content using a tool; generate summaries and skill suggestions based on user input and job descriptions.
- Real-time Resume Preview: Generate a real-time resume preview that updates as the user inputs data.
- Template Selection and Customization: Enable users to select from various resume templates (professional, modern, creative) and customize them.
- Export Options: Provide options to export the resume in multiple formats, including PDF and HTML.

## Style Guidelines:

- Primary color: Indigo (#4F5D75) to convey professionalism and competence. Conveys a tech-friendly vibe without being trendy.
- Background color: Off-white (#F0F0F0) to provide a clean and readable surface.
- Accent color: Slate Blue (#7CA1B4) to highlight key elements and calls to action.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Use Font Awesome icons to visually represent different sections and actions.
- Use a clear, two-column layout to separate the input form and the resume preview.
- Subtle transitions and animations to enhance user experience when navigating steps or generating content.
